Instrucciones

Abre este archivo y añade clases de Tailwind a los elementos de abajo para estilizarlos. No escribas CSS, solo clases.

  • Ponle al H1 un tamaño de texto grande (text-4xl), negrita (font-bold) y color azul (text-blue-600).
  • Ponle al párrafo un color de texto gris oscuro y aumenta un poco el interlineado.
  • Estiliza el botón: fondo azul, texto blanco, bordes redondeados, padding.
